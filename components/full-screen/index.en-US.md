## API

### FullScreen

| Property | Description             | Type        | Required | Default | Alternative |
| -------- | ----------------------- | ----------- | -------- | ------- | ----------- |
| enabled  | 全屏开关(受控属性)      | boolean     | 是       | --      | --          |
| isBody   | 是否是整个页面          | boolean     | 否       | `false` | --          |
| target   | 需要控制全屏的 DOM 元素 | HTMLElement | 否       | --      | --          |
| onClose  | 全屏关闭回调            | function    | 否       | --      | --          |
